Responsive Layouts

What are responsive layouts? They are used to respond to the size of the website, so if i was to shrink the size of my website the columns dont go all over the place instead they stack. Try it.

In my example the responsive layout will start to happen when the webpage is lower than 800px. The width is set to make sure my columns still take up the page rather than staying the same size and being too big.

CSS Box Model

  • The main conent (middle) is where the text and images appear.
  • The Paddiing clears an area around the main content.
  • The border is around the padding and main content.
  • The Margin is empty space around the border which has no colour.

    • This is used to make sure the content fits on the page and so that the content on the page is seperated.
